Help Page For Weather App
1. Add Location Bookmarks by clicking bookmark button in home screen
2. Add Location by clicking on desired location on Map by pointing marker
3. Saved Screen will display Bookmarked location list
4. Swipe Right to Left to delete a Bookmark location from Saved Screen
5. Click on Bookmark location to view weather details and next 5 day forecasts
6. In Setting screen, You can delete all the bookmarks at once and change the temperature unit
